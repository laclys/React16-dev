React16 + Redux + React-Router4 + MongoDB + Express + Socket.io Project

### Build Setup

```bash
# git clone
git clone https://github.com/laclys/React16-dev.git

# install dependencies
npm install

# Mongo Daemon
mongod --config /usr/local/etc/mongod.conf

# start Mongo
mongo

# build project
npm run build

# run project localhost:9098
npm run server

```
